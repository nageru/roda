Un esdeveniment de preservació agrega metadades sobre accions, concretament documentant quins objectes afecta i quins agents humans o de programari van intervenir. La documentació de les accions que modifiquen un objecte és fonamental per mantenir la procedència digital, un element clau de l'autenticitat. Accions que creen noves les relacions o alterar les relacions existents són importants per explicar aquestes relacions. Fins i tot accions que no alteren res, com la validesa i les comprovacions d'integritat dels objectes, poden ser importants per registrar-les amb finalitats de gestió.
