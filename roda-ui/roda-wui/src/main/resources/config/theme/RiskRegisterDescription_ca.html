El registre de riscos enumera tots els riscos identificats que poden afectar el dipòsit. Hauria de ser el més complet possible per incloure-hi totes les amenaces identificables i, en general, contenen una probabilitat estimada que es produeixi cada esdeveniment de risc, la gravetat o el possible impacte del risc i el seu moment probable o freqüència prevista. La mitigació del risc és el procés de definició d'accions per millorar oportunitats i reduir les amenaces als objectius del repositori.
