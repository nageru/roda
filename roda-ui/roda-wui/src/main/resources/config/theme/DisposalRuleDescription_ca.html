Les regles d'eliminació són un conjunt de requisits que determinen el calendari d'eliminació de cada entitat intel·lectual en aquest repositori. Les regles d'eliminació es poden aplicar en qualsevol moment per tal de mantenir la coherència del repositori. Les normes d'eliminació també es poden aplicar durant el procés d'ingestió. Les normes d'eliminació tenen una propietat prioritària en la qual s'executen. Si un registre no està cobert per cap de les normes, no s'associarà a un calendari d'eliminació.
