El servei de gestió d'usuaris permet al gestor de dipòsits crear o modificar les credencials d'inici de sessió per a cada usuari del sistema. Això servei també permet al gestor definir grups i permisos per cadascun dels usuaris registrats. Els gestors també poden filtrar usuaris i grups que es mostra actualment fent clic a les opcions disponibles a panell lateral dret. Per crear un nou usuari, feu clic al botó "Afegeix usuari". Per crear un nou grup d'usuaris, feu clic al botó "Afegeix grup". Per editar un usuari o grup existent, feu clic a un element de la taula següent.
