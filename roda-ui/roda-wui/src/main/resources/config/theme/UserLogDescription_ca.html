Els registres d'esdeveniments són fitxers especials que registren esdeveniments significatius que succeeix al repositori. Per exemple, es manté un registre cada vegada que a l'usuari inicia sessió, quan es fa una descàrrega o quan es fa una modificació a un fitxer de metadades descriptius. Sempre que ocorren aquests esdeveniments, el El repositori registra la informació necessària al registre d'esdeveniments per activar-lo futura auditoria de l'activitat del sistema. Per a cada esdeveniment el següent es registra la informació: data, component implicat, mètode del sistema o funció, objectes objectiu, usuari que ha executat l'acció, la durada de l'acció i l'adreça IP de l'usuari que ha executat l'acció. Els usuaris poden filtrar esdeveniments per tipus, data i altres atributs seleccionant les opcions disponibles al panell lateral dret.
