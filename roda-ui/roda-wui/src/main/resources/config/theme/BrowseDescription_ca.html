El catàleg és l'inventari de tots els articles o registres que es troben el repositori. Un registre pot representar qualsevol entitat d'informació disponible al repositori (per exemple, llibre, document electrònic, imatge, base de dades exportació, etc.). Els registres normalment s'agreguen en col·leccions (o fons) i posteriorment organitzat en subcol·leccions, seccions, sèries, fitxers, etc. Aquesta pàgina enumera totes les agregacions de primer nivell del fitxer repositori. Podeu desglossar les subagregacions fent clic a qualsevol dels elements de la taula següent.
