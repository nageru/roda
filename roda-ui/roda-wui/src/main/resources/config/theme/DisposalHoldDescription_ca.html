Les retencions d'eliminació són ordres legals o altres ordres administratives que interromp el procés normal d'eliminació i evita la destrucció de una entitat intel·lectual mentre la retenció d'eliminació està vigent. On es troba la bodega d'eliminació associat a un registre individual, impedeix la destrucció d'aquest registre mentre el la retenció d'eliminació continua activa. Un cop s'aixeca la retenció d'eliminació, l'eliminació del registre el procés continua.
