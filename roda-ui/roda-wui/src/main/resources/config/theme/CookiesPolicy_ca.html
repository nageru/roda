Política de galetes

Per fer que aquest lloc funcioni correctament, de vegades col·loquem dades petites fitxers anomenats galetes al vostre dispositiu.

Què són les galetes?

Una galeta és un petit fitxer de text que un lloc web desa al vostre ordinador o dispositiu mòbil quan visiteu el lloc. Habilita el lloc web per recordar les vostres accions i preferències (com ara inici de sessió, idioma, mida de lletra i altres preferències de visualització) durant un període de temps, de manera que no cal que tornis a introduir-los sempre que vinguis tornar al lloc o navegar d'una pàgina a una altra.

Com fem servir les galetes?

Fem servir diferents tipus de cookies en aquest lloc web. Alguns o tots de les cookies identificades a continuació es poden emmagatzemar al vostre navegador.

Categoria d'ús

Exemple

Preferències
Aquestes cookies permeten als nostres llocs web recordar informació que canvia la manera com es comporta o es veu el lloc, com ara el vostre idioma preferit o la regió on et trobes. Per exemple, recordant la teva regió, a El lloc web pot oferir-vos informes meteorològics locals o notícies de trànsit. Aquestes galetes també us poden ajudar a canviar el text mida, tipus de lletra i altres parts de les pàgines web que podeu personalitzar.
La pèrdua de la informació emmagatzemada en una galeta de preferències pot fer que L'experiència del lloc web és menys funcional però no hauria d'impedir-ho treball.
Seguretat
Fem servir galetes de seguretat per autenticar usuaris, evitar l'ús fraudulent de les credencials d'inici de sessió i protegir les dades dels usuaris de persones no autoritzades.
Processos
Les galetes de procés ajuden a fer el lloc web treballar i oferir serveis que el visitant del lloc web espera, com ara navegar per pàgines web o accedir a àrees segures del lloc web. Sense aquestes cookies, el lloc web no pot funcionar correctament.
Estat de la sessió
Els llocs web sovint recullen informació sobre com interactuen els usuaris amb un lloc web. Això pot incloure les pàgines els usuaris visiten més sovint i si els usuaris reben missatges d'error determinades pàgines. Utilitzem aquestes anomenades "galetes d'estat de sessió" per ajudar millorem els nostres serveis, per tal de millorar la navegació dels nostres usuaris experiència. El bloqueig o la supressió d'aquestes galetes no representarà lloc web inutilitzable.
Analítica
Google Analytics és l'eina d'anàlisi web gratuïta de Google que principalment ajuda els propietaris de llocs web a entendre com interactuen els seus visitants el seu lloc web. També estableix cookies al domini del lloc que ets visitant. Utilitza un conjunt de galetes amb noms com '__utma' i '__utmz' per recopilar informació de forma anònima i informar de les tendències del lloc web sense identificar visitants individuals. Més informació sobre Analítica galetes i informació de privadesa.

Com controlar les galetes

Podeu controlar i/o eliminar les galetes com vulgueu; per obtenir més informació, vegeu http://aboutcookies.org. Podeu esborrar totes les galetes que ja estan al vostre ordinador i podeu configurar la majoria navegadors per evitar que es col·loquin. Si feu això, però, és possible que hàgiu d'ajustar manualment algunes preferències cada vegada que visiteu un lloc i alguns serveis i funcionalitats poden no funcionar.

Dades privades

La informació relacionada amb les galetes no s'utilitza per identificar-vos personalment i les dades del patró estan totalment sota el nostre control. Aquests Les cookies no s'utilitzen per a cap altra finalitat que no sigui la descrita aquí.
