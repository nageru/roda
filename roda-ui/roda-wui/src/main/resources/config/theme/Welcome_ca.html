Benvingut a RODA!

Un dipòsit digital de codi obert dissenyat per a la conservació

RODA és una solució de dipòsit digital que ofereix funcionalitat per a totes les unitats principals del model de referència OAIS. RODA és capaç d'ingerir, gestionar i donar accés als diferents tipus d'objectes digitals produïts per grans corporacions o organismes públics. RODA es basa en tecnologies de codi obert i és compatible amb estàndards existents com ara el sistema d'informació d'arxiu obert (OAIS), l'estàndard de transmissió i codificació de metadades (METS), la descripció d'arxiu codificat (EAD), el Dublin Core (DC) i el PREMIS (preservació). Metadades).

Compleix amb els estàndards oberts

RODA segueix estàndards oberts utilitzant EAD per a la descripció metadades, PREMIS per a metadades de preservació, METS per estructural metadades i diversos estàndards per a metadades tècniques (per exemple, NISO Z39.87 per a imatges fixes digitals).

Proveïdor independent

RODA està 100% construït a partir de tecnologies de codi obert. El Tota la infraestructura necessària per donar suport a RODA és el proveïdor independent. Això vol dir que podeu utilitzar el maquinari i Linux distribucions que millor s'adaptin a les vostres necessitats institucionals.

Escalable

La naturalesa orientada al servei de l'arquitectura de RODA permet que el sistema sigui altament escalable, permetent la distribució del processament de càrrega entre diversos servidors. A més, l'ús de Els components avançats d'indexació permeten que els serveis de descobriment de RODA siguin repartit a través de diversos servidors d'un clúster encara més rendiment.

Accions de preservació incrustades

Les accions de preservació i la gestió dins de RODA es gestiona per un mòdul d'execució de treballs. El mòdul d'execució de treballs permet gestor de dipòsits per executar tasques de preservació sobre un conjunt determinat de dades. Les accions de preservació inclouen conversions de format, suma de comprovació verificacions, informes (p. ex., per enviar automàticament SIP correus electrònics d'acceptació/rebuig), controls de virus, etc.

Autenticitat

RODA utilitza metadades de preservació (PREMIS) per crear una confiança cadena entre totes les migracions de format i verificacions de contingut. El metadades de preservació, juntament amb l'establiment de confiança de el seu entorn (ISO 16363) garanteix la fiabilitat de la servei i autenticitat dels documents digitals adjunts.

Suport per a diversos formats

RODA és capaç d'ingerir tot tipus de tipus de contingut. Els components d'acció de migració admeten la migració de documents de text, imatges ràster, bases de dades relacionals, vídeo i àudio formats normalitzats per a la conservació a llarg termini. Un mecanisme de connexió permet que RODA admeti fàcilment migracions de format addicionals.

Fa front a la naturalesa canviant ràpid de la tecnologia

El connector i el mòdul d'execució de treballs permeten fer-ho fàcilment afegir més funcionalitat al sistema (per exemple, una nova conservació accions, alertes, eines, etc.). També, orientat al servei L'arquitectura de RODA permet que els seus components siguin incrementals actualitzat, permetent un ús heterogeni de les tecnologies nivells: maquinari, sistema operatiu i aplicacions.

Control d'accés avançat

Els usuaris s'han d'autenticar abans d'accedir al repositori. Totes les accions dels usuaris es registren per a la responsabilitat futura. Permisos són granulars i es poden definir a nivell de repositori, fins i tot fins a objectes de dades individuals.

Integració amb sistemes de tercers

RODA exposa tota la seva funcionalitat mitjançant REST ben documentat API de serveis. Les biblioteques Java convenients estan disponibles a GitHub per permetre als desenvolupadors interactuar amb RODA mitjançant les seves API bàsiques. Diversos existeixen eines per crear i manipular els SIP i enviar-los Flux de treball d'ingestió de RODA.

Flux de treball d'ingesta avançat

RODA també admet la ingesta de material digital nou metadades associades en diversos formats diferents. Es proporcionen eines per permetre als productors crear SIP en els formats compatibles. El El flux de treball d'ingestió es pot personalitzar a la interfície d'usuari per fer front a l'especial polítiques i dades.

Mercat

RODA 5 va presentar el RODA Marketplace, una plataforma digital on els usuaris poden navegar i comprar complements, components i serveis de RODA. És un centralitzat ubicació on els desenvolupadors poden mostrar els seus productes i els usuaris poden fer-ho fàcilment descobrir i adquirir el programari o els serveis que necessiten.

I molt més...

RODA (Repositori de registres digitals autèntics) és una activitat a llarg termini sistema de dipòsit digital que implementa un flux de treball d'ingesta que no només valida els SIP, però també comprova el seu contingut per detectar virus, ho fa identificació del format, extreu metadades tècniques i migra el fitxer formats a substituts més "preservables".

RODA també ofereix accés a informació digital en diversos formularis com ara cercar/navegar per les metadades disponibles, així com visualització en línia i descàrrega d'originals, formats de conservació i derivats de difusió.

Les interfícies d'administració permeten als usuaris del backoffice del dipòsit gestionar fons/col·leccions i definir normes per a les actuacions de conservació. Totes les interaccions entre usuaris (humans i màquines) i el repositori es registren per motius de seguretat i responsabilitat.

RODA garanteix que les dades ingerides siguin autèntiques enregistrant PREMIS metadades de totes les accions realitzades pel repositori, registres procedència en estàndards de metadades d'arxiu com EAD o DC, i garanteix la integritat i la disponibilitat mitjançant un seguiment freqüent de les dades i assegurant-se que no s'ha manipulat.

Llicència
